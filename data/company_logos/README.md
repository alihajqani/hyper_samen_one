# Company logos

Put company logos here, named by **company name** exactly as it appears in the
inventory's «شرکت» column — e.g. `نستله.png`, `Unilever.jpg`.

Supported extensions: `.png .jpg .jpeg .webp .bmp .gif` (the stem is matched
case-insensitively). The product detail popup shows the matching logo, or a
"no image" placeholder when none is found.

Override this folder with the `COMPANY_LOGOS_DIR` entry in `.env`.
