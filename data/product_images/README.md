# Product images

Put product pictures here, named by **barcode** — e.g. `6190.png`, `6291234567890.jpg`.

Supported extensions: `.png .jpg .jpeg .webp .bmp .gif` (the stem is matched
case-insensitively). The product detail popup shows the matching image, or a
"no image" placeholder when none is found.

Override this folder with the `PRODUCT_IMAGES_DIR` entry in `.env`.
