# Frontend assets

Place the Persian UI font here as **`Vazirmatn-Regular.ttf`**.

- Download: https://github.com/rastikerdar/vazirmatn (OFL-licensed).
- If the font is absent, the app falls back to the system default font (still RTL/Persian),
  so this file is optional for development.
- It is bundled into the executable by `tools/build_exe.py` (see the `build-windows` skill).
